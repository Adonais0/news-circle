# News-circle Application

- Run front-end: `npm install`
- `ionic serve`
- Run backend `cd backend`
    - Create virtualenv `virtualenv -p python3 venv`
    - Install dependencies `pip install -r requirements.txt`
    - `cd server`
    - `python3 manage.py runserver`
